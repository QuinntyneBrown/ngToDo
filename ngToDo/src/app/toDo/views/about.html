Overview

The purpose of this project is to

  • Implement an application lifecycle in Angular 1.3 that can be easily migrating to Angular 1.4+
  • Implement an pluggable Persistance and storage layer that can connect to Microsoft Excel or Entity Framework.

Pluggable ORM Layer: Leveraging Microsoft Excel

The boundary between the API and the Data Infrastructer is ToDoRepository. The ToDoRepository implements the IRepository interface.

Application Lifecycle

Angular 1.4+ implements a new component lifecycle where a controller has a few methods it can implement to opt in to the component lifecycle.
